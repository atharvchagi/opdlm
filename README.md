# OPDLM Research Project Page

Project page for "Data-Efficient Autoregressive-to-Diffusion Language Models via On-Policy Distillation".

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
```

## Deploying to Vercel

Import the GitHub repository into Vercel. Vercel will detect Next.js automatically and use:

- Build command: `npm run build`
- Install command: `npm install`
- Output: managed by Next.js
